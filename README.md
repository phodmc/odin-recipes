### First HTML Project
Hi! In this project, am practicing all what I've learnt so far in HTML and that includes:
- HTML Boilerplate, with the HTML Version `<!DOCTYPE html>`, the root `<html>` tag, the `<head>` tag and it's child elements and finally the 
  `<body>` tag
- Heading Tags and all their levels `<h1>` to `<h6>`
- Paragraph Element `<p>`
- Anchor Elements `<a>` and it's `href` attribute
- The self-closing `<img src=''/>` element
- Unordered `<ul>`, Ordered `<ol>` and the list `<li>` elements
